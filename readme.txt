A plugin that hooks into natively generated WordPress template code and converts class names to BEM notation.

== Installation ==
1. Upload \\\"test-plugin.php\\\" to the \\\"/wp-content/plugins/\\\" directory.
1. Activate the plugin through the \\\"Plugins\\\" menu in WordPress.
1. Place \\\"do_action( \\\'plugin_name_hook\\\' );\\\" in your templates.
